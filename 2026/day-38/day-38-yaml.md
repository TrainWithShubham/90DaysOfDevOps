1. What are the two ways to write a list in YAML?

A. First:

hobbies:
  - workout
  - movies
  - reading

Second:

hobbies: [workout, movies, reading]

2. We cannot use tabs in .yaml files. It takes it as a wrong format

3. When to use | and > in yaml?

A. "|" preserves all new lines and breaks exactly as they are writtenin file.
   ">" shows line breaks in yaml file but appears as a single line in output.

4. What you learned?

- true/false are booleans, "true" is a string
- Always use spaces instead of tabs in yaml files
- | and > are used for different purposes in the file and how it appears in output
