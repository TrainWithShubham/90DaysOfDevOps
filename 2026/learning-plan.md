# 90 Days of DevOps – Learning Plan

## Current Level

* Student
* New to DevOps and Cloud concepts

## My Understanding of DevOps & Cloud

* DevOps is a culture that connects development and operations
* Focuses on automation, faster delivery, and reliability
* Cloud helps deploy and manage applications without physical servers

## Why I Am Learning DevOps & Cloud

* To gain real-world, job-ready skills
* To understand how applications work in production
* To improve my career opportunities in IT
* To move beyond only writing code

## My 90-Day Goals

1. Build strong fundamentals of Linux, Git, and networking
2. Learn CI/CD pipelines and automation basics
3. Deploy a real application using cloud and DevOps tools

## Core DevOps Skills to Build

* Linux troubleshooting and system basics
* CI/CD pipeline creation and understanding
* Deployment and monitoring fundamentals

## Time Commitment
* Weekdays: 2–2.5 hours per day
* Weekends: 4–6 hours

## Consistency Plan
* Daily learning without skipping days
* Learn in public by sharing progress on LinkedIn and GitHub
* Focus on consistency over perfection
